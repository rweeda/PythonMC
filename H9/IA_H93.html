De elementen van een lijst staan op volgorde. Ieder element van een lijst kun je pakken via een index (of positie). Het eerste element in een lijst heeft index 0:

  • index 0: eerste element
  • index 1: tweede element
  • index 2: derde element
  • ...
  • index (lengte-1): laatste element

Stel we maken een lijst: boodschappenlijst = [ "appels", "brood", "melk" , "kaas"].

boodschappenlijst "appels" "brood" "melk" "kaas"
index 0 1 2 3

In deze lijst staat "appels" op index 0, "brood" op index 1, ... , en "kaas" op index 3 (oftewel, lengte van de lijst -1).
