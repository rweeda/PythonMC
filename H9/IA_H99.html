Met + kun je twee lijsten achter elkaar 'plakken'. Het resultaat is een lijst die de elementen bevat van beide lijsten. Je moet het resultaat in een nieuwe variabele opslaan.

Opdracht 9.9 Gescheurde boodschappenlijst

Je boodschappenlijst is door midden gescheurd. Schrijf code om de twee helften weer aan elkaar te plakken. Test je code door de nieuwe lijst te printen.

eerste_helft = ["brood", "kaas", "appels"]
tweede_helft = ["chips", "cola"]
