Hoofdstuk 7 - Herhalingen met een while-loop

Computers raken niet verveeld als ze dingen moeten herhalen. In dit hoofdstuk maak je kennis met een while-loop waarmee je code kunt laten herhalen.

In dit onderwerp:

  • leer je code te laten herhalen met een while-loop;
  • leer je hoe je een stroomdiagram met een while om kan zetten in code en andersom;
  • leer je een logische vlag te gebruiken om het aantal herhalingen te laten afhangen van gebruikersinvoer;
  • leer je een logische vlag te combineren met een conditie om het aantal herhalingen te bepalen;
  • leer je een while-loop met een teller te gebruiken om te tellen hoe vaak een herhaling plaats heeft gevonden.
