Een herhaling zoals zolang.. dan.. heet in Python een while-constructie.

  • Het begint met een test (conditie) in de while.
  • Als uit de conditie False komt slaat Python het blok code dat bij de while hoort over, en gaat daaronder verder.
  • Als uit de test True (waar) komt, dan wordt het blok code onder de while uitgevoerd.
  • Wanneer het blok code uitgevoerd is, gaat Python terug naar de test na de while en test die opnieuw.
  • Als de voorwaarde nog steeds True oplevert, dan wordt het blok code nogmaals uitgevoerd. Dit wordt steeds herhaald, totdat uit de test False komt.
